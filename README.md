# OnlineRetailDB

## Project Overview
OnlineRetailDB is a SQL Server database project that simulates an online retail management system. The project includes database creation, table relationships, sample data insertion, joins, aggregations, subqueries, and reporting queries for managing customers, products, categories, orders, and sales analysis. This project was created for learning and practice purposes while exploring SQL Server database design and query concepts.

## Database Name
OnlineRetailDB

## Tables Included
- Customers
- Products
- Categories
- Orders
- OrderItems

## Features
- Database creation using SQL Server
- Primary Key & Foreign Key relationships
- Sample data insertion
- Joins and aggregations
- Subqueries and reporting queries
- Sales and customer analysis

## SQL Concepts Used
- DDL Commands
- DML Commands
- Joins
- Aggregate Functions
- GROUP BY & HAVING
- Subqueries
- ORDER BY
- TOP Clause
- Constraints
- Identity Columns

## Sample Queries Included
- Retrieve customer orders
- Calculate total sales per product
- Find top spending customers
- Retrieve popular product categories
- Find out-of-stock products
- Monthly order analysis
- Revenue by category
- Highest priced products
- Frequently ordered products

## Tools Used
- SQL Server
- SQL Server Management Studio (SSMS)

## How to Run
1. Open SQL Server Management Studio (SSMS)
2. Run the SQL script file
3. Database and tables will be created automatically
4. Execute the sample queries
